LDAP in docker

In your superset git root, edit: vim docker/docker-bootstrap.sh add:

 apt-get install libldap2-dev python-dev libssl-dev python3-dev libsasl2-dev -y

on line 34, so it looks like:

if [ -f "${REQUIREMENTS_LOCAL}" ]; then
  echo "Installing local overrides at ${REQUIREMENTS_LOCAL}"
  apt-get install libldap2-dev python-dev libssl-dev python3-dev libsasl2-dev -y
  pip install -r "${REQUIREMENTS_LOCAL}"
else
  echo "Skipping local overrides"
fi

Note

This file is not in the git ignore, so you might have to remove it to get get a new master pull and add it back 😒

REQUIREMENTS_LOCAL

This file is found here: vim ./docker/requirements-local.txt and should contain python-ldap

LDAP can now be configured.

Basic ldap config

# Active directory connection
from flask_appbuilder.security.manager import AUTH_LDAP
AUTH_TYPE                   = AUTH_LDAP
AUTH_LDAP_SERVER            = "ldap://ad.lan:389"
AUTH_LDAP_SEARCH            = ""
AUTH_LDAP_BIND_USER         = ""
AUTH_LDAP_BIND_PASSWORD     = ""
AUTH_LDAP_UID_FIELD         = "sAMAccountName"
AUTH_LDAP_USE_TLS           = False
AUTH_USER_REGISTRATION      = True
AUTH_USER_REGISTRATION_ROLE = "Public"
AUTH_ROLES_MAPPING = {
    "": ["Admin"],
    "": ["Alpha"],
    "": ["Gamma"],
}
AUTH_ROLES_SYNC_AT_LOGIN    = True
