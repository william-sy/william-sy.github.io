Firewalld

Add a service to firewwalld

firewall-cmd --zone=public --add-service=logstash

To make this permanent after reboot

firewall-cmd --zone=public --permanent --add-service=logstash

The logstash file needs to be present in order for this to work

SSH example:

/usr/lib/firewalld/services/ssh.xml

<?xml version="1.0" encoding="utf-8"?>
<service>
  <short>SSH</short>
  <description>Secure Shell (SSH) is a protocol for logging into and executing commands on remote machines. It provides secure encrypted communications. If you plan on accessing your machine remotely via SSH over a firewalled interface, enable this option. You need the openssh-server package installed for this option to be useful.</description>
  <port protocol="tcp" port="22"/>
</service>
